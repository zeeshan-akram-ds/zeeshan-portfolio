## Technical Architecture

Built as a lightweight, highly optimized static single-page application (SPA) designed to load instantly with complete responsive integrity across mobile and desktop displays.

- **Markup & Layout:** Pure HTML5 semantic foundations paired with fluid CSS custom property mapping (`clamp()` dynamic scaling, native CSS Grid multi-column structures).
- **Theme Controls:** Integrated data-attribute dark/light mode toggle with persistent background contrasts ensuring custom presentation deck visuals remain readable under both themes.
- **Interactivity:** Lightweight vanilla JavaScript driving modal navigation, theme persistence via browser storage, and IntersectionObserver API metrics triggers.
- **Asset Integration:** Links direct users straight to production deliverables—such as native GitHub high-resolution PDF rendering engines—avoiding uncurated raw source code friction.

## Featured Flagship Diagnostics

### 01. Retail Discount Profitability & Margin Leakage Audit
- **Objective:** Proving whether store discounting acts as a promotional tool or a permanent, unmanaged pricing floor.
- **Results:** Quantified an active $97 profit gap per discounted line across ~224,000 transaction records, mapping a centralized corporate habit that compressed blended margins down to 52%.
- **Deliverables Embedded:** End-to-end PostgreSQL schema briefs, multi-page ReportLab executable PDFs, and high-level slide deck narratives.

### 02. SaaS Contract Churn & Revenue Risk Deep-Dive
- **Objective:** Isolating active subscription drop-offs concealed by blended MRR averages.
- **Results:** Quantified a $139K recurring baseline leak and traced 86.9% of total lifecycle loss directly to month-to-month contract design exposures.

### 03. eCommerce Logistics Root-Cause Churn Audit
- **Objective:** Reframing a high single-purchase churn rate from a assumed product failure to an operational delivery timeline issue.
- **Results:** Proved delayed fulfillment windows permanently destroyed customer account lifecycles, consuming top-line customer acquisition budgets.
