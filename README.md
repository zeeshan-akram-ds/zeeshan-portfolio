# Zeeshan Akram | Data Analyst Portfolio Website

Welcome to the repository for my personal portfolio website. This site serves as a centralized hub for my Data Analytics case studies, Business Intelligence dashboards, and technical articles.

My focus is on transforming raw databases into clear, executive-level business stories.

---

## Live URL
https://zeeshan-portfolio-swart.vercel.app/

---

## Built With

This website is a lightweight, static single-page application (SPA) built to be fast, responsive, and accessible.

- **HTML5** — Semantic structure and SEO optimization  
- **CSS3** — Custom styling, Flexbox/Grid layouts, and responsive media queries  
- **Vanilla JavaScript** — DOM manipulation and interactive UI elements  

### Libraries Used

- **Typed.js** — Dynamic typing effect in the hero section  
- **AOS (Animate On Scroll)** — Smooth scrolling animations  
- **Font Awesome** — Vector icons  

---

## Key Features

- **Data Analyst Focused:** Specifically curated to highlight SQL queries, EDA in Python, and Power BI/Streamlit dashboards, filtering out unrelated noise.  
- **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop viewing experiences.  
- **Direct Resume Download:** One-click access to my updated Data Analyst resume.  
- **SEO Optimized:** Configured with proper meta tags, Open Graph (OG) tags, and Google Analytics tracking.

## Local Development

If you want to run this project locally on your machine:

### Clone the repository

```bash
git clone https://github.com/zeeshan-akram-ds/zeeshan-portfolio.git
``` 

### Navigate to the project directory

```bash
cd zeeshan-portfolio
```


### Run the project

Open `index.html` in your browser, or use a live server extension (like VS Code Live Server) to view changes in real-time.
